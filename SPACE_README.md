# Redrob Candidate Ranker — Sandbox Demo

Upload a `candidates.jsonl` file and get a ranked shortlist for the Senior AI Engineer
role at Redrob AI — scored using a multi-signal weighted model (skills, career history,
behavioral signals, education, location). No API calls, no GPU required.

Try it with `data/sample_candidates.jsonl` (50 candidates) included in this repo.

See `README.md` in the repo root for full methodology.
